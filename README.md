# Study Universe 🌌

Study Universe is a creative interactive web project built using HTML, CSS, JavaScript and jQuery.

It combines multiple study-focused modules in one platform:

- 🧠 Memory Rooms — organize and review notes  
- 🌦 Study Mood — generate study methods based on mood  
- 🧬 Future Planner — simulate outcomes from habits  
- ⏱ Focus Mode — timed focus sessions  
- 🗺 Learning Map — visualize learning progress  

## Technologies Used
- HTML5  
- CSS3  
- JavaScript  
- jQuery  
- LocalStorage  

## Purpose
Created as a front-end project exploring interactive UI design, creativity and productivity tools.

## Run Project
Open `index.html` in a browser.
